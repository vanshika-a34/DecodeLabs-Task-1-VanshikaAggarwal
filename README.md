# 🚀 Data Science Internship | DecodeLabs

Welcome to my Data Science Internship repository! This project showcases the work I completed during my internship at **DecodeLabs**, focusing on data preprocessing, feature engineering, and supervised machine learning pipelines.

---

## 📊 Project 1: Data Cleaning & Feature Engineering
**Goal:** Transform raw, chaotic e-commerce data into a mathematically clean dataset optimized for machine learning.

### 🛠️ Key Requirements
*   **Data Integrity:** Handled missing values through identifying nulls and applying categorical filling (e.g., mapping missing Coupon Codes to **'N/A'**).
*   **Outlier Management:** Identified and neutralized statistical anomalies using the **Interquartile Range (IQR)** method to enhance model stability.
*   **Feature Engineering:** Architected **4+ new predictive features**, including `Month` (seasonal trends), `CartConversionRate`, `ItemsAbandoned`, and `RevenuePerCartItem`.

### 🧠 Key Skills
*   `Python` | `Pandas` | `NumPy`
*   `Statistical Analysis` | `Outlier Detection`
*   `Feature Engineering`

---

## 🛡️ Project 2: Supervised Learning (Fraud Detection Pipeline)
**Goal:** Build and fine-tune a robust classification model to identify fraudulent transactions within highly imbalanced datasets.

### ⚙️ Key Requirements
*   **Imbalance Handling:** Implemented **SMOTE (Synthetic Minority Over-sampling Technique)** to balance class distributions effectively.
*   **Model Benchmarking:** Trained and compared multiple algorithms including **Logistic Regression** and **Random Forest** using `Scikit-Learn`.
*   **Performance Evaluation:** Prioritized high-stakes metrics by discarding simple "Accuracy" in favor of **Precision**, **Recall**, and **ROC-AUC**.
*   **Pipeline Optimization:** Leveraged `Scikit-Learn` pipelines for reproducible and scalable workflows.

### ⚡ Key Skills
*   `Classification Algorithms`
*   `Imbalanced Data Handling`
*   `Supervised Learning`
*   `Hyperparameter Tuning`
